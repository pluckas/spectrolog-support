# Spectro Log

Spectroscopy target manager for macOS

---

## About

SpectroLog is a macOS application for managing and organising astronomical spectroscopic targets and observations.

It allows users to store target information, log observations, and review observational history in a structured and searchable way.

The application is designed for local use and does not require user accounts or cloud-based storage.

## Privacy Policy

Spectro Log does not collect, store, transmit, or share any personal data. The application may access network services only to retrieve astronomical object coordinates from the SIMBAD astronomical database operated by the Centre de Données astronomiques de Strasbourg (CDS), France. No user information is transmitted or stored. Spectro Log does not use analytics, tracking technologies, advertising frameworks, or third-party data collection.
